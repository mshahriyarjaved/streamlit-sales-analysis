# Sales Analysis Dashboard (Streamlit)

##  Overview
This is a simple interactive dashboard built using Python Streamlit.
It helps analyze sales data dynamically using filters and charts.

---

##  Tech Stack
- Python 
- Streamlit 
- Pandas 
- NumPy 

---

##  Features
- CSV file upload
- City & Product filters
- Real-time KPI metrics
- City-wise bar chart
- Clean interactive UI
